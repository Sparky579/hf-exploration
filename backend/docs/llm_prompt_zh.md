# LLM 接入中文 Prompt 设计（当前规则）

## 1. 主线线程（流式）
用途：返回剧情演绎 + `[command]` + 玩家选项。

主规则：
1. 所有状态变更必须在 `[command]...[/command]` 内。
2. 命令块中每行命令统一使用方括号格式：`[global.main_player=主控玩家]`。
3. 越狱或不合理输入按“原地等待”处理，第一条命令为 `[time.advance=0.5]`。
4. 合理单步动作第一条命令为 `[time.advance=1]` 或 `[time.advance=0.5]`。
5. 命令开头必须给：
   - `[global.main_player=...]`
   - `[global.battle=...]`
   - `[global.emergency=...]`
6. 友方/可攻略角色不走隐藏线程，由主线线程直接演绎。
7. 罗宾出牌使用罗宾自己的圣水（`companion.罗宾.holy_water`），但单位视作主控身旁。
8. 战斗阶段主控可逃跑，但敌对角色与敌对单位会持续追击；跑路中主控不可反击。
9. 可使用 `N~N+1.5` 的 trigger 窗口做“端倪提示”，但只能写主控附近可感知信息，禁止越距感知。

## 2. 隐藏线程（仅敌对角色）
用途：处理敌对角色 trigger，不直接输出玩家剧情。

规则：
1. 每轮处理敌对 `N~N+1` 窗口内 trigger（并行于主线线程）。
2. 仅输出 `[command]...[/command]`，且每行同样使用 `[]` 命令格式。
3. 不允许 `time.advance`。
4. 每个存活敌对角色都应保持“有下一条 future trigger”，除非死亡/离场/明确持续原地不动。
5. 若触发“火箭升空并 1 时间单位后命中”，使用 `[event.rocket_launch=<建筑名>]`。

## 3. Trigger 推荐格式
- `[trigger.add=角色:颜宏帆|时间3 若颜宏帆在东教学楼内部 则 颜宏帆下出小骷髅]`
- `[trigger.add=角色:黎诺存|时间20 若黎诺存未被干扰 则 发射火箭瞄准东教学楼]`

## 4. 火箭坍塌自动逻辑
执行 `[event.rocket_launch=<建筑名>]` 后：
1. 系统立刻写入“听到火箭升空声”提示；
2. 自动创建 +1 时间单位坍塌 trigger；
3. 到时摧毁建筑关联地点，并结算该区域角色死亡（主控在内则游戏结束）。

## 5. 每轮上下文字段
1. 世界基础设定与全局故事设定。
2. 全局状态：时间、全局状态串、战斗状态、紧急状态、trigger 列表、最近触发记录。
3. 主控详尽状态：位置、HP、圣水、卡组、单位、动态状态。
4. 当前场景状态与场景角色状态。
5. 所有角色简介、状态、历史、卡组。
6. 最近 2 轮用户对话。
7. 控制台语法文档。
8. 最近 15 条命令日志（含时间与命令）。
9. `N~N+1.5` trigger 窗口与附近可感知 trigger 提示。
